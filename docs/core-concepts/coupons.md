# Coupons (Promotions)

**Coupons** allow you to offer discounts, either fixed or percentage-based, to your clients. This feature is very useful for promotional events, affiliate programs, or maintaining customer loyalty.

## Creating a New Coupon

In Billmora, promotional code configurations can be made very specific. You are not just creating random codes, but you can also restrict the code to specific products, specific users, or a specific duration.

### Field Glossary

| Field | Description |
| :--- | :--- |
| **Code** | Enter a unique code that customers will use to redeem this coupon. |
| **Type** | Select whether the coupon provides a percentage discount or a fixed amount discount. |
| **Value** | Enter the discount value. For percentage, enter a percent amount. For fixed amount, the value uses the default currency and will be automatically converted to other currencies using the base rate. |
| **Billing Cycles** | Select specific pricing types for this coupon. Leave empty to apply to all billing cycles. |

### Limitations and Usage Rules

| Field | Description |
| :--- | :--- |
| **Maximum Uses** | Set the maximum number of times this coupon can be used globally. Leave blank for unlimited uses. |
| **Maximum Uses Per User** | Set the maximum number of times a single user can use this coupon. Leave blank for unlimited uses. |
| **Start Date** | Select the date and time when the coupon becomes valid. Leave blank for immediate validity. |
| **Expiration Date** | Select the date and time when the coupon expires. Leave blank for no expiration. |

### Applicability

| Field | Description |
| :--- | :--- |
| **Applicable Packages** | Select the packages that this coupon can be applied to. Leave blank to allow all packages. |
| **Applicable TLDs** | Select the Top Level Domains (TLDs) that this coupon can be applied to. Leave blank to allow all TLDs. |
