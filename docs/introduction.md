# Introduction

**Billmora** is a free, open-source Billing Management, Operation, and Recurring Automation platform built for hosting providers and service businesses.

It acts as a self-hosted alternative to platforms like WHMCS or Blesta, giving you full control over your billing, client management, and server provisioning without any recurring licensing fees.

## Tech Stack

Billmora is built for performance and maintainability leveraging current technologies:
- **Backend:** Laravel 12.x (PHP 8.2+)
- **Frontend Stack:** Laravel Blade, Livewire 3.x, AlpineJS, Tailwind CSS, Vite
- **Database:** MySQL 8.0+ / MariaDB 10.4+ *(Fully tested & supported. PostgreSQL is experimental)*
- **Background Jobs:** Laravel Queues (Redis strongly recommended, or Database)
